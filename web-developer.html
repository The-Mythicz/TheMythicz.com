Web developer

Hyderabad, Telangana, India

Apply

Minimum Qualifications:

  • A bachelor’s degree in computer science, information technology, or a related field.
  • Experience in HTML, CSS, JavaScript, Python, PHP, Ruby, Java, or Node.js
  • Knowledge of frameworks like React, Angular, or Vue.js for front-end, and Django, Flask, or Express.js for back-end.
  • Familiarity with Git and platforms like GitHub or GitLab.
  • Ability to create websites that work on various devices using techniques like mobile-first design and CSS frameworks
  • Understanding how to design environments that are visually engaging and serve the gameplay or narrative.
  • Understanding tools for debugging and testing
  • Familiarity with SQL or NoSQL databases
  • Ability to troubleshoot and resolve issues.

Preferred Qualifications:

  • Familiarity with task runners/build tools (Webpack, Gulp, etc.). Knowledge of CI/CD pipelines and DevOps practices.
  • Experience with cloud platforms like AWS, Azure, or Google Cloud.
  • Understanding of design principles and experience with tools like Figma or Adobe XD.
  • Familiarity with securing web applications against common vulnerabilities (e.g., XSS, CSRF, SQL Injection).
  • A strong portfolio showcasing past projects, especially those demonstrating complex problem-solving and innovation.
  • Willingness to learn new technologies and stay updated on industry trends.

Responsibilities:

  • Develop and manage forums, chat systems, or social networking integrations for player communities.
  • Work with the design team to create and maintain a consistent user experience across all platforms.
  • Collaborate with the back-end team to integrate APIs and databases.
  • Optimize websites for speed and scalability.
  • Implement security measures to protect user data.
  • Stay updated on emerging technologies and best practices.
  • Work with the marketing team to create landing pages and promotional materials.
  • Contribute to the overall design and functionality of the website.
  • Provide technical support to users and team members.
  • Participate in code reviews and testing.
  • Ensure the website is accessible and responsive.
  • Collaborate with other developers and designers to create a seamless user experience.
  • Stay updated on industry trends and best practices.
